The sum of the primes below $10$ is $2 + 3 + 5 + 7 = 17$.
Find the sum of all the primes below two million.

Answer:Â 142913828922

---
```python
n = 2000000

st = [False] * (n + 1)
primes = []
sum = 0
for i in range(2, n + 1):
    if not st[i]:
        primes.append(i)
        sum += i
    j = 0
    while primes[j] * i <= n:
        st[primes[j] * i] = True
        if i % primes[j] == 0:
            break
        j += 1
print(sum)
```
