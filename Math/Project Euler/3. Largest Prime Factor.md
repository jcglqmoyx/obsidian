The prime factors of $13195$ are $5, 7, 13$ and $29$.
What is the largest prime factor of the number $600851475143$?

Answer:Â 6857

---
```python
n = 600851475143
i = 3
res = 0
while i * i <= n:
    while n % i == 0:
        n //= i
        res = max(res, i)
    i += 2
if n > 1:
    res = max(res, n)
print(res)
```
